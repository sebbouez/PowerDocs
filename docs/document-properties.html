Document properties and variables names

Variables scopes

There are 3 types of scopes for variables in PowerDocs:

Scope name

Description

Document

The variables with Document scope are available anywhere in the content and templates

Chapter

The variables with Chapter scope are only available in the current chapter

Headers

The variables with Headers scope are only available in the Headers and Footers

General properties

The General panel contains default properties of your document. Once set, you can use them in your document (in Markdown content and theme templates) using their variable names.

Property

Description

Variable name

Variable scope

Author

Type the author of the document

%DocumentAuthor

Document

Service

If you work in a company, it may be interesting to provide your service name

Enterprise

%DocumentCompany

%DocumentCopyright

%DocumentVersion

Chapter properties

Variable name

Variable scope

Description

%ChapterName

Chapter

The current chapter title

Examples

You can use the current chapter name in the content or in the theme template like that:

The current chapter name is %ChapterName

Customize headers and footers

Variables for PDF outputs

Variable name

Variable scope

Description

%CurrentTitle

Headers

The current running H1 title

%PageNum

Headers

The current page number

%PageCount

Headers

The total count of pages in the exported document

Examples

To display "Page 1 on 10" in the footer, use the custom expression:

Page %PageNum on %PageCount
