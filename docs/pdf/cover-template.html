# Custom Cover template

To specify a template in the your output plan file, you can use the template property like this:

...
- title: My document title
  template: mydoccover.tpl
  type: Cover
...

By specifying a .tpl file in the template property, PowerDocs will load the file and apply your own markup.
Please note that the path to the .tpl file is relative to the project root folder, without leading slash.

The .tpl file works like an .html file, except that you must not specify classical html tags. Also, you must not put content outside of the root tag such as style or external libraries.

# Document cover page

The template must be included in the documentCover tag.

<documentCover>
<img src="templates/logo.png"/>
<small>MY COMPANY</small>
<p>{tpl.title}</p> <!-- This will display the "title" property of the plan item -->
</documentCover>

# Chapter cover page

The template must be included in the chapterCover tag.

<chapterCover>
<img src="templates/logo.png"/>
<small>MY COMPANY</small>
<p>{tpl.title}</p> <!-- This will display the "title" property of the plan item -->
</chapterCover>

# Templates variables

VariableDescription
{tpl.title}Displays the current page number
{tpl.documentVersion}Displays the number of pages in the output document
{tpl.author}Displays the value of Author
