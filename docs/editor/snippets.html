# Working with snippets

You may need to have portions of content to be included at sevaral places of your documentation. To avoid copy/paste, you can use the snippets feature.

By default, your project will be created with a ".snippets" folder. This is where you can create your .md files that will be included in other places.

You can use the inline Markdown tag [snippet(....)] with a relative path from the ".snippets" folder.

Example

[snippet(snippet1.md)]

Or you can use the drag/drop hint feature:

  1. Drag the snippet file from the Project Explorer panel to any place on your document
  2. The editor will create an hyperlink to the file by default
  3. Click the little bulb that appear next to the dropped text
  4. Select Insert as snippet
