PowerDocs online help Help

Edit theme

The document rendering is processed using HTML markup.

To edit the document theme files, follow these steps:

  1. Click the Tools menu

  2. Select the Theme editor menu

Default blocks CSS classes

As described on this page, PowerDocs provides default blocks to use.

All the blocks are rendered as DIV elements and have the pwd-block CSS Class.

Block type

CSS Class Name

Info

.pwd-block.block-info

Warning

.pwd-block.block-warning

Alert

.pwd-block.block-alert

Note

.pwd-block.block-note

Stop

.pwd-block.block-stop

Templates for PDF outputs

The theme must contain the following files:

Template file name

Usage

chapter-cover.tpl

This template file is used to render each chapter cover page

document-cover.tpl

This template file is used to render the cover page of the document

If a template is missing at render time, PowerDocs will replace it with a default template.
