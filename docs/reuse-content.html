Reuse pieces of content

It's sometimes important to repeat information across several pages. In PowerDocs you can create topics and import them at several places.

By using this technic you only have to edit your shared topics once and modifications will apply automatically everywhere they are imported.

Create a reusable topic

Create a new Chapter in your document. Mark the new chapter as "Exclude" (it won't appear when you export your document). Create a new topic in the excluded Chapter. Edit your topic.

Import Markdown from an external file

  1. Open the topic

  2. Click Insert > Insert content from external file

  3. Browse the file

  4. Click OK
